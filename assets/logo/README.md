# Container Composer Logo

This directory contains the official logo files for Container Composer.

## Design Concept

The logo combines two key elements:

1. **Containers** - Stacked boxes representing Docker containers, showing the hierarchy and organization of services
2. **Musical Composition** - Musical staff lines and notes representing the "composer" aspect - orchestrating and harmonizing different services together

The design symbolizes how Container Composer helps you orchestrate your Docker services like a conductor leading a symphony.

## Files

### Main Logos

- **`logo.svg`** - Full logo (400x400) for light backgrounds
- **`logo-dark.svg`** - Full logo (400x400) for dark backgrounds
- **`banner.svg`** - Horizontal banner (1200x300) with text
- **`icon.svg`** - Simplified icon (200x200) for small spaces

### Usage Guidelines

#### In Documentation
Use `banner.svg` at the top of documentation pages.

#### In README
Use the banner or main logo at the top of the README file.

#### As Favicon
Use `icon.svg` converted to `.ico` or `.png` format for favicons.

#### In Social Media
Use `logo.svg` for profile pictures and social media posts.
